​ 第一篇博客，就用来记录如何通过hexo快速发布一篇博文并且推送到github仓库中吧。

一、常规的方法

  • step1:

    cd /basic_path/blog/

    hexo new “article title”

    通过typora或者其他的md编辑器，编辑、保存博文。

  • step2:

    hexo server

    启动本地服务，预览 localhost:4000

  • step3:

    hexo clean && deploy

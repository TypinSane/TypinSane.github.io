Lecture 1.线性等式的几何学意义

发表于2025-09-07|更新于2025-09-11

看MIT 18.06的线代课，做个学习笔记。

2 dimensions

线性等式

A ⋅ x = b

Row picture（行图像）

把两个等式分别画在平面直角坐标系中， 两条直线相交的的点，就是方程的解。

Column picture(列图像)

首先把上面的等式转为** Linear combination of column** 可以把矩阵拆解成两个向量（此例的维度是2），两个向量根据x，y进行缩放之后（黑色的向量进行2被缩放，变成灰色），进行平移首尾相连（途中红色和灰色），最终得到（0，3）向量。缩放的倍率就是我们的未知数的解。

我理解这里的Row picture和column picture 是指的从一个矩阵的行来进行理解还是从列来进行理解。

  • 从行来理解就能把举证和变量组合成中学时期学过的二元一次方程组，再画出两条直线相交的图像。
  • 从列来理解就是转换为向量，再通过向量的缩放，平移，组合来进行理解。

这里获得了x、y的解，此时的解是矩阵A 产生（0，3）这个向量的解，如果b对应的不是一个固定的值，而是任意的一个向量，那么x和y的解的集合是什么？

很容易想象， 两个向量可以任意缩放，平移与组合，只要这两个向量不平行，产生的结果就能铺满整个平面。

3 dimensions

线性等式

Row picture（行图像）

在维度为3的情况下，每一个线性等式都在三维空间中构成了一个平面.

  • 对于第一个方程组，因为与z的取值无关，所以构成的平面就是2x-y=0这条直线沿着z轴扫过的平面。这个面垂直于xy平面。
  • 对于第二个方程组，就是所有xyz取值所构成的平面。（为什么三个未知数的等式的所有解构成一个平面呢？一个个人比较愚笨的理解：先把z放在一边，还是看-x+2y=0， 这里x和y构成了xy平面上的一条直线，此时可以把（-z+1）看作一个整体D，即一个可能为任意值的变量。如果我们从俯视角来看，这个D的变化对xy构成的直线造成的影响是在坐标轴上向着某个方向进行平移，不会产生旋转，如果此时再把z轴的变化给加进去，就能观察到这条直线在向z轴方向移动的时候， 在xy这个“水平”的方向在发生平移，每一次平移产生的直线，都和上一次保持平行，并且z轴的变化对水平方向的影响是线性的，所以就构成了一个平面。与第一个方程组中与z无关的区别也能够看出来了，就是这里产生的平面与xy水平面不垂直）
  • 第三个方程与第一个同理。

三个平面相交的点，就是这个方程组的解

Column picture(列图像)

Linear combination of column

三个向量组合出来的结果能否填满整个三维空间？

如果三个向量没有在同一个平面上，那么就能够填满整个三维空间。

与二维空间的原理相同，两个向量组合出的结果就能构成一个平面，如果第三个向量脱离了这两个向量，那么就可以使这个平面在三维空间中平移，扫过整个空间，从而填满。但是如果第三个向量也在这个平面之中，那么组合的结果也只能在这个二维平面中，无法超出。

线性方程组是否有解？

如果三个向量不在同一个平面，那么肯定有解；如果在同一个平面，b如果在平面上，有解，否则无解。
